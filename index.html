Kodi LG Repository v1.0.0

LG remote enhancements for Kodi: OSD on Up/Down during playback & voice search.

Install Instructions
  1. Add source in Kodi File Manager
    Settings → File Manager → Add source → https://unrefundable.github.io/kodi-lg/ → name it Kodi LG
  2. Install repository from zip
    Settings → Add-ons → Install from zip file → select Kodi LG source → click repository.kodi.lg-1.0.0.zip
  3. Install the add-on from repository
    Settings → Add-ons → Install from repository → Kodi LG Repository → Services → Kodi LG → Install
